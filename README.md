### 1. Stop profit.
买入后一直持有的A与设置有止盈点的C之间发生的故事。     ₍₍Ϡ(੭•̀ω•́)੭✧⃛  
初始资产一百万，随机挑选一个基金，再随机挑选一个时间点，买入并持有两百个交易日后卖出，每次交易收取百分之七的佣金。  
区别是A在买入时间点买入后一直持有，直到满两百个交易日。  
而C设有止盈点，达到止盈点后就卖出，卖出后再随机买入一支基金，如此循环往复，直到过完两百天。  
当然，对于C的这种两百个交易日内的买卖，不收取佣金。  

重复一万次，得到如下（资产-重复次数）结果图，其中A用红色标记，C用蓝色标记。  
其中C有五条策略，分别对应设置百分之十到百分之九十的止盈点。  
可以看到，C的止盈点设置的越高，收益越高，最高能略微超过A。  
![stop_profit_result](https://user-images.githubusercontent.com/32698786/128802941-fc495619-d7d1-421f-994f-891df01f2a04.png)  

得到如下结论：  
**结论一：设置小幅度的止盈点只会降低收益。**     
**结论二：设置大幅度的止盈点可能有一点点正向作用。**   
设置止盈点会降低收益，是否说明近期上涨的基金更有希望继续继续上涨？  
是否说明购买近期上涨的基金有希望获得更高收益？  
值得进一步讨论。

### 1. Automatic investment plan.
喜欢梭哈的A与喜欢定投的B之间发生的故事。   ₍ᐢ •⌄• ᐢ₎  
初始资产一百万，随机挑选一个基金，再随机挑选一个时间点，买入并持有两百个交易日后卖出，每次交易收取百分之七的佣金。  
区别是A在买入时间点一次性买入，而B在买入时间点周围分十次，每次间隔十个交易日定投买入。  
重复一万次，得到如下（资产-重复次数）结果图，其中A用红色标记，B用蓝色标记。  
其中B有三条策略，由上到下，分别对应A投入前B就结束定投，A投入时B刚好完成一半定投，A投入后B才开始定投。  

得到如下结论：  
**结论一：就数学期望来讲，基金随便买，一次买一年，次数多了稳赚不赔。**  
**结论二：就绝对收益来讲，一次性买入与定投相差无几。**    
如果说定投有缺点的话，便是在定投的过程中没有享受到还未投入的资金所带来的收益。  

此外，该方法还可用于估算基金投资的期望收益。只要看每次操作需要扣除多少比例的本金才能维持一万次后资产总量保持不变即可。  
可以粗略算出，按照A的策略，即：  
随机挑选一个基金，再随机挑选一个时间点，买入并持有两百个交易日后卖出。  
在不计算佣金的情况下，完成一次交易后预期的收益约为9.7%。

### 1. days_probability
![image](https://user-images.githubusercontent.com/32698786/123397576-275ba580-d5d5-11eb-8b94-a7b76edca9f0.png)

### 2. the_big_down
![image](https://user-images.githubusercontent.com/32698786/123398954-a7ced600-d5d6-11eb-89d3-40e13855bc9f.png)  
在每次证券指数发生单日大幅度（超过2%）上涨下跌前，通常经过一段相对较长的平缓上升时期，再加一段相对较短的快速下跌时期。
### 3. image_analysis

### 4. the_support
https://user-images.githubusercontent.com/32698786/123956686-8c503a80-d9dd-11eb-93aa-6fa06ef02574.mp4

历史上的高点和低点会不会影响股价明天的走势？  
当今天的股价达到了近历史上的一个高点，明天突破这个价位的概率会变大还是变小？  
这里给出一个统计上的答案：到达历史高点并向上突破的概率为0.518，到达历史低点并向下突破的概率为0.437。  
ps: 这里用到的数据量较小（到达历史高点27次，到达历史低点32次），且改变参数后结果差异较大。
